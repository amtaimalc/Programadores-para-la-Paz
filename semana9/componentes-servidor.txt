Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar programas y gestionar procesos, archivos y dispositivos.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Allí se definen las rutas, las respuestas y el comportamiento de la aplicación.

Puerto:
Es un punto de comunicación que permite que el servidor reciba solicitudes. En esta actividad el servidor escucha en el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del sistema. Ayudan a identificar errores y supervisar el funcionamiento del servidor.
