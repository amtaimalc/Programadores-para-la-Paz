# Programadores para la Paz - MT

## Project Overview
This is an educational repository for the "Programadores para la Paz" (Programmers for Peace) course. It contains text-based course activity files organized by week.

## Project Structure
- `semana1/` - Week 1 course activities
  - `algoritmo-semana1.txt` - Algorithm exercise: steps for sharing reliable information online
  - `preguntas-semana1.txt` - Quiz answers for week 1
  - `reflexion-semana1.txt` - Weekly reflection on responsible internet use and Git

## Notes
- This is a text-only project with no frontend, backend, build system, or dependencies.
- Content is in Spanish.
- No web server or workflow is needed.
