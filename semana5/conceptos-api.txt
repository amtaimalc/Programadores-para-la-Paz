La ruta POST /reporte permite recibir información enviada por el usuario mediante una solicitud POST. El servidor toma el mensaje enviado y responde con un objeto JSON indicando que el reporte fue recibido correctamente. Esta ruta podría servir en una aplicación comunitaria para recibir reportes ciudadanos, alertas, solicitudes de ayuda o información importante enviada por miembros de la comunidad de manera organizada.
